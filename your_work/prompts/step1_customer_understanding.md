# Step1: 顧客理解

## 目的
身近な課題から顧客像を作り、行動を分析し、改善すべき課題を特定する

## プロンプト

```
あなたはユーザーリサーチの専門家です。
私が抱えている課題は「[ここに自分の課題を記入]」です。

以下の手順で分析してください：

1. ペルソナ作成
   この課題を抱える典型的な顧客のペルソナを作成してください。
   - 年齢、家族構成、職業、趣味
   - 1日の簡単なスケジュール
   - この課題に直面する心理的・状況的必然性

2. 行動分析
   この顧客が目的達成のために取る行動を時系列で整理してください。
   - 各行動の所要時間
   - 各行動のコスト（金額/労力）
   - 各行動時の感情（ポジティブ/ネガティブ）

3. 課題の特定
   行動の中で最も改善インパクトが大きい課題を3つ特定してください。
   - 削減できるリソース × 頻度で評価
   - 各課題を「なぜ○○せずに□□ができないのか？」の形で表現

表形式でまとめてください。

**重要**: 結果を `ideation/step1_persona.md` に保存してください。
```

## レビュー・修正フェーズ

生成された内容を確認し、以下の観点で自分で考えて修正してください：

- [ ] ペルソナは自分が想定する顧客像と合っているか？
- [ ] 行動分析は現実的か？抜けている行動はないか？
- [ ] 課題Top3は本当に重要な課題か？優先順位は適切か？

**必要に応じて `ideation/step1_persona.md` を直接編集してください。**

AIの出力をそのまま受け入れず、自分の頭で考えることが重要です。

## 使い方

1. 上記プロンプトをコピー
2. `[ここに課題を記入]` の部分を自分の課題に置き換え
3. Amazon Q Developerチャットに貼り付け
   - VSCode: 左側のAmazon Qアイコンをクリック
   - （オプション）CLI: `q chat`
4. 出力結果を確認し、必要に応じて追加質問

## 期待されるアウトプット

- ペルソナ（顧客像）
- 行動の時系列マップ
- 改善すべき課題Top3

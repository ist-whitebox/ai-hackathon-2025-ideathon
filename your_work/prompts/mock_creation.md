# モック作成（25分）

## 目的
Strands Agentsを活用してポン出しモックを作成する

## モックの優先順位

1. 🥇 **Python + Flask/FastAPI + 画面 + Strands Agents処理**（推奨）
2. 🥈 **Node.js/Next.js + 画面 + 処理**
3. 🥉 **HTML + JavaScript + Python**
4. 🏅 **HTML + JavaScript（入力→出力）**

## プロンプト（推奨版）

```
PR/FAQの内容を基に、以下の構成でモックを作成してください：

技術スタック：
- Python（Flask または FastAPI）
- 簡単なHTML画面
- 処理部分はStrands Agentsを活用
- ローカルで動作確認できること

要件：
1. ランディングページ（サービス説明）
2. 主要機能の体験UI（入力フォーム）
3. Strands Agentsによる処理実行
4. 結果表示画面

実装手順も含めて教えてください。
```

## プロンプト（簡易版）

```
PR/FAQの内容を基に、以下の構成でモックを作成してください：

技術スタック：
- HTML + JavaScript
- 簡単な入力→出力の動作

要件：
1. ランディングページ
2. 入力フォーム
3. ダミーの処理結果表示

実装手順も含めて教えてください。
```

## 使い方

1. Step3で作成したPR/FAQを確認
2. 推奨版または簡易版のプロンプトを選択
3. Amazon Q Developer (`q chat`) に貼り付け
4. 生成されたコードを実行
5. ローカルで動作確認

## 重要

- **ポン出しレベルでOK**: 完璧を求めない
- **動けばOK**: デザインは気にしない
- **時間がない場合**: 簡易版に切り替える
- **モックなしでもOK**: PR/FAQだけでも提出可能

## トラブルシューティング

### Strands Agentsでエラーが出る
- 前回のサンプルコードを参考に
- 簡易版（HTML+JS）に切り替え

### 時間が足りない
- PR/FAQだけで提出
- 発表時に「構想」として説明
